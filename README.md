# AI Booster

Telegram bot that delivers AI-driven trading signals and localized rationales (EN/AR) with risk-managed entries, stops, take-profits, and position-sizing suggestions for beginner traders.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
